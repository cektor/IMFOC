1- Open Terminal.
2- sudo chmod +x install.sh
3- bash install.sh
4- Open Nemo File Manager
5- From the top MenuBar, click Edit to Open Extensions (Alt+P)
6- Activate the IMFOC+NemoPytyhon Plugin from the Plugins List if it is Inactive.
7- Right Click on an Image File with the Mouse and Click Convert to Different Format.
8- Select the format you want to convert.
9-Conversion Completed.
10-The converted file will come to the directory where your image file is located.
